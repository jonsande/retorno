config set audio <on|off>
config set ambientsound <on|off>
config set theme <linux|amber|green|ice>

Purpose
- Inspect and adjust operator sensory and terminal presentation settings.

Operational behavior
- `config show` displays the current sensory and terminal presentation profile.
- `config set lang <en|es>` selects the language in which manuals and terminal messages are presented to the operator.
- `config set audio <on|off>` enables or suppresses the full neural audio feed.
- `config set ambientsound <on|off>` enables or suppresses the ambient layer while keeping the rest of the audio feed available.
- `config set theme <linux|amber|green|ice>` changes the active display palette for the operator terminal projection.

Usage notes
- Language switch affects manuals, messages, and localized file preference.
- Audio preferences remain in effect until changed again.
- Display palette preference remains in effect until changed again.

Related commands
- `status`, `man <topic>`, `cat <path>`
