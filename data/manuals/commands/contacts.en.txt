nav map contacts

Command status
- `contacts` exists as a compatibility alias.
- Canonical command: `nav map contacts`.

Accepted aliases (hidden in `help`)
- `contacts` -> `nav map contacts`
- `nav contacts` -> `nav map contacts`

Dependencies
- Depends only on `core_os` (same as all `nav map *`).
- No dependency on `sensors` or `data_core`.

Output format
- Header: `=== CONTACTS ===`
- Shows known contacts, estimated distance, and `route/no_route`, `visited/unvisited` flags.
- While in transit it uses known-contact cache (`in transit; using cached known contacts`).
