install

Command status
- Legacy command removed from active workflow.
- Module operations are under `drone` command family.

Operational behavior
- Use `drone install <drone_id> <module_id>` for installation.
- Use `drone uninstall <drone_id> <module_id>` for uninstallation.
- Install behavior resolves by module scope:
  - `scope=ship`: installs into ship module stack.
  - `scope=drone`: installs into target drone module stack (bay operation).

Related commands
- `drone install <drone_id> <module_id>`, `drone uninstall <drone_id> <module_id>`, `module inspect <module_id>`, `modules`
