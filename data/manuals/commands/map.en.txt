map path <node_id>

Command status
- `map` is no longer a canonical mapping command.
- It is preserved only as a compatibility alias for `nav map path <node_id>`.

Equivalent alias
- `map path <node_id>` -> `nav map path <node_id>`

Notes
- `map ship`, `map graph`, and `map graph <node_id>` were removed.
- Use `ship sectors`, `nav map graph`, and `nav map graph <node_id>`.
