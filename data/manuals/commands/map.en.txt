map path <node_id>

Command status
- `map` is retained only for backward compatibility.
- Canonical mapping namespace is `nav map ...`.

Equivalent form
- `map path <node_id>` -> `nav map path <node_id>`

Removed legacy forms
- `map ship`
- `map graph`
- `map graph <node_id>`

Operational guidance
- Use `ship sectors` for internal ship topology.
- Use `nav map graph` and `nav map graph <node_id>` for world graph operations.

Related commands
- `nav map sectors`, `nav map graph [node_id]`, `nav map routes`, `nav map contacts`
