module inspect <module_id>

Purpose
- Inspect one module definition from catalog and current runtime placement.

Operational behavior
- `module inspect` works for both ship and drone modules.
- Output includes:
  - scope (`ship` or `drone`),
  - effects,
  - drawbacks,
  - slot cost (`drone` modules),
  - runtime presence (inventory / installed on ship / installed on drones).
- Catalog inspection does not require module to be currently installed.

Related commands
- `modules`, `drone install <drone_id> <module_id>`, `drone uninstall <drone_id> <module_id>`, `man <module_id>`
