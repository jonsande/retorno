install

Estado del comando
- Comando legado retirado del flujo activo.
- Las operaciones de modulos viven en la familia `drone`.

Comportamiento operativo
- Usa `drone install <drone_id> <module_id>` para instalar.
- Usa `drone uninstall <drone_id> <module_id>` para desinstalar.
- La instalacion se resuelve por scope del modulo:
  - `scope=ship`: instala en la pila de modulos de nave.
  - `scope=drone`: instala en la pila de modulos del dron objetivo (operacion de bahia).

Comandos relacionados
- `drone install <drone_id> <module_id>`, `drone uninstall <drone_id> <module_id>`, `module inspect <module_id>`, `modules`
