map path <node_id>

Estado del comando
- `map` se conserva solo por compatibilidad retroactiva.
- El espacio canónico de cartografía es `nav map ...`.

Forma equivalente
- `map path <node_id>` -> `nav map path <node_id>`

Formas heredadas retiradas
- `map ship`
- `map graph`
- `map graph <node_id>`

Guía operativa
- Usa `ship sectors` para topología interna de nave.
- Usa `nav map graph` y `nav map graph <node_id>` para cartografía del grafo mundial.

Comandos relacionados
- `nav map sectors`, `nav map galaxy [sector|local|regional|global]`, `nav map graph [node_id]`, `nav map routes`, `nav map contacts`
