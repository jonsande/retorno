map path <node_id>

Estado del comando
- `map` ya no es comando canónico de cartografía.
- Se mantiene solo como alias de compatibilidad para `nav map path <node_id>`.

Alias equivalente
- `map path <node_id>` -> `nav map path <node_id>`

Notas
- `map ship`, `map graph` y `map graph <node_id>` fueron retirados.
- Usa `ship sectors`, `nav map graph` y `nav map graph <node_id>`.
