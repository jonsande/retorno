SYSTEM SUMMARY — power_core
Ref: OS-MAN/SYS/CORE-PWR

`power_core` is RETORNO's primary electrical generation source.

Function
- Converts reactor resources into ship-usable electrical output.
- Provides baseline generation for all downstream subsystems.

Output profile by state
- OFFLINE: 0% generation
- CRITICAL: 25% generation
- DAMAGED: 50% generation
- LIMITED: 75% generation
- NOMINAL: 100% generation

Health-state thresholds (default)
- `health <= 0.00` -> OFFLINE
- `health < 0.35` -> CRITICAL
- `health < 0.60` -> DAMAGED
- `health < 0.85` -> LIMITED
- `health >= 0.85` -> NOMINAL

Operational impact
- Core degradation rapidly reduces electrical headroom and increases deficit/brownout risk.
- Low generation propagates into lower `Q`, service blocks, and accelerated system stress.

Recovery pattern
- reduce non-critical load,
- repair `power_core`,
- rebuild battery reserve and stabilize `Q`.

Related commands
- `diag power_core`, `power status`, `drone repair <drone_id> power_core`, `alerts`
